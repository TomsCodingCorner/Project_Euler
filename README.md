# TomsCodingCorner — Project Euler in Python

This repository contains my Python solutions and explanations for Project Euler problems, written mainly in Jupyter Notebooks.

The goal of this project is not only to solve the problems, but to build a clear programming portfolio that shows mathematical reasoning, algorithmic thinking, clean Python code and technical communication.

## Current progress

Sequential progress: **20 / 100**

## Why I am doing this

Project Euler is a good way to train the fundamentals behind software engineering, data science and AI:

- breaking down abstract problems into smaller steps
- translating mathematical ideas into Python
- comparing simple and optimized approaches
- writing readable code
- explaining technical solutions clearly
- building consistency through a long-term coding project

This repository is part of my personal programming portfolio.
